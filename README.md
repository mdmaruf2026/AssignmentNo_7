# KeenKeeper

KeenKeeper is a sophisticated contact management dashboard built to help users maintain and organize their social and professional connections. Inspired by high-end Figma designs, this project focuses on delivering a seamless and visually appealing user interface with robust navigation and state management.

## Technologies Used

- **React.js**: For building a dynamic, component-based single-page application.
- **Tailwind CSS**: For implementing a modern, utility-first design system with precise styling.
- **Lucide React**: For a consistent and scalable iconography system.
- **React Router**: For handling smooth, client-side navigation without page reloads.

## Key Features

### 1. High-Fidelity Interactive Navigation
The application features a smart navigation system that provides instant visual feedback. Using **Active State Highlighting**, the current page is marked with a dark-green pill-shaped button style (`#1A4D2E`). This ensures a superior User Experience (UX) by keeping the user oriented at all times within the app's structure.

### 2. Precision Branding & Typography
Unlike standard projects that use static images for logos, KeenKeeper implements a **Pure CSS Dual-Tone Branding**. The logo is engineered using extra-bold typography (`font-[900]`) and negative letter spacing (`tracking-tighter`) to match strict Figma specifications. It uses a specific color split—Dark Slate (`#1e293b`) and Forest Green (`#1A4D2E`)—ensuring the brand identity is sharp on all screen resolutions.

### 3. Optimized Sticky Layout Management
To ensure maximum usability, the dashboard utilizes an advanced **Sticky UI Layout**. The navigation bar is locked at the top of the viewport (`sticky top-0`) with a high-priority stacking order (`z-50`). This allows users to access navigation controls regardless of how far they scroll down long lists of data, all while maintaining a clean, borderless, and minimalist aesthetic.
